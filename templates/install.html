Step 3: Install Ubuntu

Install Ubuntu on both laptops following these rules and then click next.:

  • Use only trustworthy guides such as this one: https://tutorials.ubuntu.com/tutorial/tutorial-install-ubuntu-desktop#0
  • Never connect the DISCONNECTED laptop to any wifi network and disable the wifi card in Ubuntu as soon as possible.
  • Delete and erase all existing partitions using the Disk utility that you can access by selecting "Try Ubuntu" before installing Ubuntu.
  • Use your SMALL external hard drive to create a bootable Ubuntu drive
