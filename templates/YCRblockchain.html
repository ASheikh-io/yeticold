Yeti Bitcoin Cold Storage

Select Test Only or Trustworthy Blockchain Data

This blockchain data is hosted on google drive and should download within an hour. It is only appropriate for testing. If you select this blockchain data you should start setup from step 1 before using real money on this device.

This blockchain data is fully trustworthy, but will take between 4 hours and 2 weeks to download. You can also select the date of the oldest deposit made into your wallet to save on disk space. Yeti will prune blockchain data before this date. If you have plenty of disk space you can leave this date blank and Yeti will store the entire blockchain.
